# Java Pong Game

A classic Pong game built using Java Swing.

## Features

- Two-player gameplay
- Real-time score tracking
- 20-second game timer
- Winner announcement
- Restart game using the R key
- Collision detection
- Ball bouncing mechanics

## Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)

## Controls

### Player 1
- W → Move Up
- S → Move Down

### Player 2
- ↑ Arrow Key → Move Up
- ↓ Arrow Key → Move Down

### Other Controls
- R → Restart Game

## Project Structure

```text
Main.java
GameFrame.java
GamePanel.java
Paddle.java
Ball.java
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/uday-kasu/java-pong-game.git
```

2. Open the project in your Java IDE

3. Compile and run:

```bash
Main.java
```
